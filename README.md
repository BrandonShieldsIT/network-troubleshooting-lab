# Network Troubleshooting Lab

## Overview
This project simulates common network issues and demonstrates troubleshooting techniques used in IT support roles.

## Scenarios
- DNS misconfiguration
- Incorrect IP settings
- Network connectivity failures

## Tools Used
- ipconfig
- ping
- nslookup

## Key Tasks Performed
- Diagnosed and resolved DNS issues
- Identified and corrected IP configuration errors
- Restored network connectivity

## Skills Demonstrated
- Network Troubleshooting
- TCP/IP Fundamentals
- Problem Solving
